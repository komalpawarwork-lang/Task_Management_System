# 📋 Task Management System

The Task Management System is a simple and interactive web application built using HTML, CSS, and JavaScript. It helps users efficiently manage daily tasks by allowing them to create, update, and track task progress.

## 🚀 Features

- ➕ Add, Edit, and Delete Tasks
- 📊 Track Task Status (Pending, Completed)
- ⏰ Set Deadlines for Tasks
- 🔍 Filter and Search Tasks
- 💾 Data persistence using Local Storage
- 📱 Responsive and user-friendly UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
- Local Storage (for data saving)

## 💡 Project Objective

The goal of this project is to build a lightweight task management system that improves productivity and demonstrates core JavaScript concepts like DOM manipulation and local storage.
